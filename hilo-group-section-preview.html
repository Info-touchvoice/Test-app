Group page structure (from APK):
• Header — purple gradient bg, 3 sub-tabs (Related / Popular / Discover), search icon
• Popular — promo banner, Family/Rank/CP leaderboard cards, then two sub-sections:
  → Popular tab: Video Match bar, quick-play games (Ludo, UNO, Sheep, Snake), hot live group rooms
  → New tab: scrollable list of newly created groups only (LatestGroupAdapter, layout item_group_popular1.xml)
• Related — my groups carousel, create group card, Recently/Joined filter, recent & recommended groups
• Discover — country flags, featured country with top-3 users, gift wall, global broadcast, activities grid, events list
• Bottom nav — Group tab highlighted purple (#9036FF)

Layout files: fragment_home_room.xml, new_view_group_popular.xml, view_my_all_group.xml, view_wrapper_discover.xml, item_group_popular1.xml
