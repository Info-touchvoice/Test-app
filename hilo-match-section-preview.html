Match page structure (from APK):
• Fragment — HomeMatchingFragment in fragment_home_matching.xml
• Planets globe — PlanetsView + PlanetCalculator: users on an invisible 3D sphere (autoScrollMode="uniform", scrollSpeed="3.0", radiusPercent="0.8"), drag rotation, depth-based scale/alpha, pulse rings (view_planet.xml)
• Card grid (2 columns, asymmetric heights):
  → Fast match (120dp, cyan gradient) — starts random video matching pool
  → Masked Chat (86dp, pink) — opens StrangeMatchActivity
  → Broadcast (86dp, purple) — global broadcast chat
  → Watch Youtube together (120dp, blue) — co-watch feature
• Hidden cards — Sphinx (game 19), Slots (14), Fishing (20) shown only when server returns them in matchGameList
• While matching — full-screen in_matching_pool.svga + "Matching randomly for you，Don't leave" + Exit match button
• Match found — view_matching_pending.xml overlay: avatar, countdown ring, accept / refuse
• Layout files — fragment_home_matching.xml, include_match_big_item.xml, include_match_min_item.xml, view_planet.xml
